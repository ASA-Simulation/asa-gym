ASAGym: Gym environments for RL training using ASA as a simulator
=================================================================

## Environment setup

We recommend that you use a virtual environment:

    $ sudo apt install python3.10-venv

    $ python3 -m venv .venv

    $ source .venv/bin/activate

    $ pip install -e .
